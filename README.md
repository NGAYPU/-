#실시간 대선 후보 얼굴 교체 시뮬레이터 

#프로젝트 개요

- 제가 진행한 프로젝트는 Python의 OpenCV와 Dlib을 활용하여, 웹캠을 통해 사용자 얼굴을 실시간으로 인식하고, 사전에 준비된 한국의 21대 주요 대선 후보 3인의 얼굴과 교체하는 프로그램입니다. 

#기획 배경 

- 9주차 Computer vision basic 온라인 강의를 보고 관련된 프로젝트를 진행해보고싶다는 생각을 했고, 시각적 흥미를 더하기 위해 실시간 얼굴 교체라는 주제를 선정했습니다. 

#주요 기능
- 실시간 얼굴 인식: 웹캠을 통해 사용자의 얼굴을 프레임 단위로 실시간 감지합니다.
- 다중 후보 전환: 키보드 입력을 통해 실시간으로 후보자의 얼굴을 선택하여 교체할 수 있습니다.
- 정밀한 얼굴 교체: Dlib의 68개 랜드마크를 활용하여 사용자 얼굴에 후보자의 얼굴을 합성합니다.


#기술구현 과정

1. 핵심 포인트 추출: Dlib의 shape_predictor_68_face_landmarks.dat 모델을 사용하여 후보 이미지와 실시간 웹캠에서 사용자 얼굴의 68개의 핵심 포인트를 추출합니다. 
2. 분할: 68개 랜드마크 중 얼굴 외곽선을 기준으로 얼굴 영역을 수백 개의 작은 삼각형으로 분할합니다.
3. 변환: 후보 얼굴의 각 삼각형 영역을 잘라내어, 사용자 얼굴의 해당 삼각형 위치와 모양에 맞게 변환합니다.
4. 블렌딩 :OpenCV의 cv2.seamlessClone() 기능을 사용하여 변환된 얼굴 조각을 사용자 얼굴에 합성할 때, 합성 이질감을 최소화합니다.

#문제 해결 과정 및 한계

- 이 프로젝트는 기술 구현 외에도 까다로운 환경 설정을 극복하는 과정이 있었습니다. Dlib 라이브러리의 C++ 기반 빌드 환경 문제를 미리 컴파일된 휠 파일을 사용하여 해결했고, 가상 환경을 구축했습니다. 
- 또한 실행 결과를 보면 알 수 있듯이 일정 범위를 벗어나면 합성이 풀리고 원래 사용자의 얼굴로 돌아온다는 점에서 향후 이 부분에 대한 보완이 필요할 것으로 보입니다!!
